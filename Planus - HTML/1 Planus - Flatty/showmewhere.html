Show Me Where was created with Node.js and Ionic, an HTML5 hybrid mobile app framework. This application uses the BetterDoctor API and the UberX API to recommend a doctor based on your injury, as well as displaying the time and cost that will take to get to your doctor via Uber. This project was built within a 24 hours time frame for the Health 2.0 Code-a-thon.
